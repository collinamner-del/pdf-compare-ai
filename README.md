# PDF Compare AI

Compare two PDF documents using AI-powered analysis.

## Features
- 📄 PDF text extraction
- 🔍 Detailed side-by-side comparison
- ✨ AI-powered summary of differences
- 🚀 Full-stack deployment ready

## Tech Stack
- **Backend:** Flask + PyMuPDF + OpenAI
- **Frontend:** React + Axios
- **Deployment:** Render (backend) + Vercel (frontend)

## Local Development

### Backend
```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend
```bash
cd frontend
npm install
npm start
```
